# Bereoth Properties and Homes Investment Limited

Bereoth is a real estate platform that allows users across different locations to browse properties, connect with company consultants, and inquire or purchase land online from the comfort of their homes.

## 🚀 Features

- Property listings with details and images  
- User consultation request and contact workflow  
- Admin dashboard to manage property listings  
- Responsive design for mobile and desktop  
- RESTful API integration for data handling

## 🛠 Tech Stack

- **Frontend:** React, JavaScript (ES6+), HTML5, CSS3  
- **Backend:** Node.js, Express.js (if applicable)  
- **Database:** PostgreSQL (if applicable)  
- **API:** RESTful services

## 👨‍💻 What I Built

- Developed scalable and reusable React components for property listings and user interactions  
- Built responsive frontend interfaces for cross-device usability  
- Implemented an admin dashboard to reduce manual listing updates by 50%  
- Structured and documented RESTful APIs for property and user data management  
- Optimized performance using lazy loading and code splitting  
- Applied accessibility best practices (a11y) and cross-browser compatibility  
- Troubleshot frontend issues and improved overall application stability

## 📈 Impact

- Streamlined property management for the company  
- Improved user experience for property seekers across locations  
- Reduced onboarding time for developers through clear documentation and structured APIs
